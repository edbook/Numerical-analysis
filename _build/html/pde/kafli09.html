Jaðargildisverkefni fyrir sporgerar (e. elliptic) hlutafleiðujöfnur¶

Almenn atriði um jaðargildisverkefni¶

Við ætlum að skoða annars stigs línulegar hlutafleiðujöfnur á svæði \(R\) í \({\mathbb R}^2\), þetta eru jöfnur á forminu

\[\begin{split}\begin{gathered} A(x,y)\frac{{\partial}^2 u}{{\partial}x^2} + B(x,y)\frac{{\partial}^2 u}{{\partial}x{\partial}y} + C(x,y)\frac{{\partial}^2 u}{{\partial}y^2} +\\ D(x,y)\frac{{\partial}u}{{\partial}x} + E(x,y)\frac{{\partial}u}{{\partial}y} + F(x,y)u = G(x,y).\end{gathered}\end{split}\]

Skilgreining¶

Annars stigs línuleg hlutafleiðujafna eins og að ofan kallast sporger (e. elliptic) ef

\[\begin{split}A(x,y)C(x,y) - B(x,y)^2 > 0,\end{split}\]

fyrir öll \((x,y) \in R\).

Sporgerar jöfnur¶

Sporgerar jöfnur eru tímaóháðar og lýsa oft verkefnum þar sem verið er að leita að jafnvægisástandi (lágmarksflötum, hitajafnvægi og þess háttar).

Laplace/Poisson¶

Við ætlum að einskorða okkur við mikilvægasta tilvikið, Laplace jöfnuna

\[\frac{{\partial}^2 u}{{\partial}x^2} + \frac{{\partial}^2 u}{{\partial}y^2} = 0,\]

og hliðraða útgáfu hennar, Poisson jöfnuna

\[\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = f(x,y).\]

Jaðarskilyrði¶

Svæðið \(R\) Til einföldunar þá munum við eingöngu skoða rétthyrninga á forminu

\[\begin{split}R = \{ (x,y) \in {\mathbb R}^2 ; a < x < b, c < y < d \}.\end{split}\]

Eins og fyrir afleiðujöfnur með jaðarskilyrði (kafli 8) þá höfum við þrjár gerðir af jaðarskilyrðum,

Dirichlet: \(u(x,y) = r(x,y)\) á \({\partial}R\).
Neumann: \(\frac{{\partial}u}{{\partial}n}(x,y) = r(x,y)\) á \({\partial}R\).
Robin: \(\alpha(x,y)u(x,y)+\beta(x,y)\frac{{\partial}u}{{\partial}n}(x,y) = r(x,y)\) á \({\partial}R\).

Hér er \({\partial}R\) jaðar svæðisins \(R\) og \(n\) er útvísandi þverill fyrir \({\partial}R\).

Þýð föll¶

Skilgreining Tvisvar sinnum samfellt diffranlegt fall \(u\) á svæði í \({\mathbb R}^2\) sem uppfyllir Laplace jöfnuna

\[\frac{{\partial}^2 u}{{\partial}x^2} + \frac{{\partial}^2 u}{{\partial}y^2} = 0,\]

kallast þýtt.

Athugasemd Tvisvar sinnum samfellt diffranlegt fall \(u\) á svæði \(R \subset {\mathbb R}^2\) er þýtt þá og því aðeins að

\[u(x) = \frac{1}{\pi r^2} \int_{B(x,r)} u(y) dy\]

fyrir öll \(x \in R\) og öll \(r\) þannig að \(B(x,r) \subset R\). Hér er \(B(x,r)\) skífan með miðju \(x\) og geisla \(r\).

Poisson jafnan í rétthyrndu svæði – Dirichlet-jaðarskilyrði¶

Netið¶

Byrjum á að skipta rétthyrningnum \(R\) í \(M \times N\) net með því að skipta bilinu \([a,b]\) í \(N\) bil og bilinu \([c,d]\) í \(M\) bil. Til einföldunar þá gerum við ráð fyrir að hlutfall hliðarlengdanna í \(R\) sé ræð tala, því þá getum við fundið \(M\) og \(N\) þannig að

\[\frac{b-a}N = \frac{d-c}M = h.\]

Þar með er hver reitur í netinu er af stærðinni \(h \times h\).

Ef við köllum látum \(x_j\) vera skiptipunktanna fyrir \([a,b]\),

\[\begin{split}a=x_0 < x_1 < x_2 < \ldots x_{N-1} < x_N =b,\end{split}\]

og \(y_k\) vera skiptipunktanna fyrir \([c,d]\),

\[\begin{split}c=y_0 < y_1 < y_2 < \ldots y_{N-1} < y_M = d,\end{split}\]

þá eru skiptipunktarnir í netinu \((x_j,y_k)\).

Ritháttur¶

Við skoðum ætlum að skoða jöfnuna

\[\frac{{\partial}^2 u}{{\partial}x^2} + \frac{{\partial}^2 u}{{\partial}y^2} = f(x,y),\]

á \(R = \{ (x,y) \in {\mathbb R}^2 ; a < x < b, c < y < d \}\) með Dirichlet skilyrðin

\[u(x,y) = g(x,y), \qquad \text{ á } {\partial}R.\]

Til að einfalda rithátt þá skrifum við

\[\begin{split}\begin{aligned} u_{j,k} &= u(x_j,y_k), \qquad j=0,\ldots,N, \quad k=0,\ldots,M,\\ f_{j,k} &= f(x_j,y_k), \qquad j=0,\ldots,N, \quad k=0,\ldots,M,\\ g_{j,k} &= g(x_j,y_k), \qquad j=0, j=N, k=0 \text{ eða } k=M,\end{aligned}\end{split}\]

og nálgunargildin eru

\[w_{j,k} \approx u_{j,k}.\]

Strjál útgáfa Poisson jöfnunar¶

Með því að nota miðsettan mismunakvóta fyrir aðra afleiðu \(u\) þá getum við skrifað

\[\begin{split}\begin{aligned} \frac{{\partial}^2 u}{{\partial}x^2}|_{x_j,y_k} &= \frac{u_{j-1,k} - 2u_{j,k} + u_{j+1,k}}{h^2} + O(h^2), \\ \frac{{\partial}^2 u}{{\partial}y^2}|_{x_j,y_k} &= \frac{u_{j,k-1} - 2u_{j,k} + u_{j,k+1}}{h^2} + O(h^2), \end{aligned}\end{split}\]

þar sem \(j = 1,\ldots,N-1\) og \(k=1,\ldots,M-1\).

Með því að setja þetta inn í

\[\frac{{\partial}^2 u}{{\partial}x^2} + \frac{{\partial}^2 u}{{\partial}y^2} = f(x,y),\]

þá fæst,

\[\frac{u_{j-1,k} - 2u_{j,k} + u_{j+1,k}}{h^2} + \frac{u_{j,k-1} - 2u_{j,k} + u_{j,k+1}}{h^2} + O(h^2) = f_{j,k}.\]

Höfðum

\[\frac{u_{j-1,k} - 2u_{j,k} + u_{j+1,k}}{h^2} + \frac{u_{j,k-1} - 2u_{j,k} + u_{j,k+1}}{h^2} + O(h^2) = f_{j,k}.\]

Skiptum \(u\) út fyrir nálgunargildin \(w_{j,k}\), hendum skekkjuliðnum og margföldum í gegn með \(-h^2\), þá fæst

\[-w_{j-1,k} - w_{j+1,k} - w_{j,k-1} - w_{j,k+1} + 4w_{j,k} = -h^2 f_{j,k},\]

fyrir \(j = 1,\ldots,N-1\) og \(k=1,\ldots,M-1\).

Fjöldi jafna¶

Hér á undan fengum við eina jöfnu fyrir hvern innri punkt netsins, samtals \((N-1)(M-1)\) jöfnur. Þetta er fjöldi óþekktra \(w_{j,k}\), því \(w_{j,k}\) á jaðrinum eru þekkt, þar er

\[w_{j,k} = g_{j,k}.\]

Við þurfum að því leysa jöfnuhneppið

\[\begin{split}\begin{aligned} -w_{j-1,k} - w_{j+1,k} - w_{j,k-1} - w_{j,k+1} + 4w_{j,k} &= -h^2 f_{j,k} \\ \text{fyrir } j=1&,\ldots,N-1, \ \, k=1,\ldots,M-1\\ \end{aligned}\end{split}\]

Uppröðun á innri punktum¶

Röðum í vigur \(\mbox{${\bf w}$}\) tölunum \(w_{j,k}\) fyrir innri punkta netsins. Förum í gegnum netið frá vinstri til hægri, byrjum á næst neðstu línunni og förum svo upp á við,

\[\mbox{${\bf w}$}= [ \underbrace{w_{1,1} \ w_{2,1} \ldots w_{N-1,1}}_{k=1} \ \underbrace{w_{1,2} \ w_{2,2} \ldots w_{N-1,2}}_{k=2} \ \underbrace{w_{1,3} \ldots \ldots}_{k=3,\ldots,M-1} ]^T.\]

Fyrri vísirinn (\(j\)) vísar til \(x\)-hnitsins \(x_j\) og seinni vísirinn (\(k\)) til \(y\)-hnitsins \(y_k\).

Jöfnurnar aftur¶

Jöfnurnar sem við leiddum út eru

\[-w_{j-1,k} - w_{j+1,k} - w_{j,k-1} - w_{j,k+1} + 4w_{j,k} = -h^2 f_{j,k}.\]

Ef \(j-1=0\), \(k-1 =0\), \(j+1=N\) eða \(k+1=M\) þá erum við á jaðrinum og þar þekkjum við gildin, þau eru gefin með \(g\).

T.d. ef \(j=1\) og \(k=1\) þá fæst

\[\begin{split}\begin{aligned} -h^2 f_{1,1} &= -w_{0,1} - w_{2,1} - w_{1,0} - w_{1,2} + 4w_{1,1} \\ &= -g_{0,1} - w_{2,1} - g_{1,0} - w_{1,2} + 4w_{1,1}\\ \end{aligned}\end{split}\]

Það er

\[- w_{2,1} - w_{1,2} + 4w_{1,1} = -h^2 f_{1,1} + g_{0,1} + g_{1,0}.\]

Jöfnurnar fyrir \(k=1\), \(j=1\), \(k=M-1\) og \(j=N-1\) þurfa því að taka mið af \(g\).

Blönduð jaðarskilyrði¶

Rifjum upp þær þrjár gerðir jaðarskilyrða sem við höfum áhuga

Dirichlet: \(u(x,y) = r(x,y)\) á \({\partial}R\).
Neumann: \(\frac{{\partial}u}{{\partial}n}(x,y) = r(x,y)\) á \({\partial}R\).
Robin: \(\alpha(x,y)u(x,y) + \beta(x,y)\frac{{\partial}u}{{\partial}n}(x,y) = r(x,y)\) á \({\partial}R\).

Hér er \({\partial}R\) jaðar svæðisins \(R\) og \(n\) er útvísandi þverill fyrir \({\partial}R\).

Neumann skilyrði eftir botninum¶

Skoðum svæðið

\[\begin{split}R = \{ (x,y) \in {\mathbb R}^2 ; a < x < b, c < y < d \}.\end{split}\]

og tilvikið þegar jaðargildin eru gefin með falli \(g(x,y)\) á hliðunum (\(x=a\) og \(x=b\)) og toppnum (\(y=d\)), og afleiðan eftir botninum er gefin með

\[\frac{{\partial}u}{{\partial}n} = \alpha(x)\]

(ath. staðsetningin þar er bara háð \(x\)).

Útvísandi þverill eftir botninum er

\[\frac{{\partial}}{{\partial}n} = -\frac{{\partial}}{{\partial}y}.\]

Mismunakvótar eftir botninum¶

Gerum eins og áður ráð fyrir því að við höfum jafna skiptingu á bilinu \([a,b]\),

\[\begin{split}a = x_0 < x_1 < \ldots < x_{N-1} < x_N = b,\end{split}\]

með billlengd \(h\), og skiptingu á bilinu \([c,d]\) með sömu billengd,

\[\begin{split}c = y_0 < y_1 < \ldots < y_{M-1} < y_M = d.\end{split}\]

Bætum við felupunktum \((x_j,y_{-1})\) fyrir \(j=1,\ldots,N-1\). Samhverfir mismunakvótar fyrir \(\frac{{\partial}u}{{\partial}n}\) gefa þá

\[- \left. \frac{{\partial}u}{{\partial}y} \right|_{x=x_j} = \alpha(x_j)\]
\[-\frac{w_{j,1} - w_{j,-1} }{2h} \approx \alpha(x_j).\]

Það er

\[w_{j,-1} = w_{j,1} + 2h\alpha(x_j)\]

Jaðarskilyrðin sett inn í Poisson-jöfnuna¶

Rifjum upp að lausnin okkar \(w_{j,k}\) á \(\Delta u = f\) á að uppfylla

\[\begin{split}\begin{aligned} -w_{j-1,k} - w_{j+1,k} - w_{j,k-1} - w_{j,k+1} + 4w_{j,k} &= -h^2 f_{j,k} \end{aligned}\end{split}\]

Eftir botninum (\(k=0\), \(j=1,\ldots,N-1\)) þýðir þetta að

\[\begin{split}\begin{aligned} - w_{2,0} - w_{1,-1} - w_{1,1} + 4w_{1,0} &= -h^2 f_{1,0} + g(0,0),\\ -w_{j-1,0} - w_{j+1,0} - w_{j,-1} - w_{j,1} + 4w_{j,0} &= -h^2 f_{j,0}, \quad {\color{gray} j=2,\ldots,N-2},\\ -w_{N-2,0} - w_{N-1,-1} - w_{N-1,1} + 4w_{N-1,0} &= -h^2 f_{N-1,0} + g(N,0) \end{aligned}\end{split}\]
